# Laudato Si’ Bot 🌍

Un chatbot basado en la encíclica *Laudato Si’* del Papa Francisco, diseñado para responder preguntas sobre ecología integral, acciones concretas y reflexiones. Incluye **autoentrenamiento** para mejorar con el feedback de los usuarios.

---

## 📌 Características
- Responde preguntas sobre la encíclica *Laudato Si’*.
- Ofrece reflexiones y citas aleatorias.
- Aprende de las correcciones de los usuarios (autoentrenamiento).
- Desplegado en la nube con Heroku.

---

## 🛠 Requisitos
- Python 3.8+
- Rasa 3.4.0
- Flask 2.3.2
- SQLite3
